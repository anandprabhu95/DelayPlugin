Delay plugin made with JUCE Framework (v7.0.12).

Parameter controls:
1) Delay Time: 0 to 2 seconds [L and R channels].
2) Feedback Gain: 0 to 1 [L and R channels].
3) Dry/Wet mix
4) Modulation frequency and amount
