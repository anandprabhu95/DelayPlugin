Delay plugin made with JUCE Framework (v7.0.12).

Parameter controls:
1) Delay Time: 0 to 2 seconds [L and R channels].
2) BPM Sync delay
3) Feedback Gain: 0 to 1 [L and R channels].
4) Dry/Wet mix
5) Modulated Delay
6) Echo Tone Control
